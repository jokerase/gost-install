# gost-install
### Centos / Ubuntu / Debian
#### 下载/运行
```
wget --no-check-certificate -O gost.sh https://raw.githubusercontent.com/jokerase/gost-install/master/gost.sh && bash gost.sh
```
#### 使用环境：python3
```

————————————————————————

 -- 6.1 设置-L参数
    -- 6.1.1 清除并重新设置 -L参数

    -- 6.2.3 不使用 -F参数
 7. 查看 当前配置
 8. 打开 配置文件

————————————————————————
```
